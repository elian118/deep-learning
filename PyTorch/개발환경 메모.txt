# Conda
- 가상환경 관리 프로그램
  - 프로그램 실행에 필요한 환경들을 마음대로/독립적으로 구성할 수 있는 공간
  - 가상환경 만드는 법
    - Python venv: 파이썬 설치 시 사용 가능
    - conda
    - poetry
    - pyenv
    - ...
  - Conda 장점
    - 목적이 Data Scientist들을 위한 라이브러리 큐레이션
    - scikit-learn, numpy, panda, matplotlib, scipy, tensorflow, pytorch
    - pip 설치 시 환경이 서로 모순될 경우 꼬일 수 있음
    - 반면에, conda는 channel이라는 개념이 있음
    - channel: 사람들끼리 모여서 최적의 라이브러리 조합을 짜줌(conda-forge)

conda는 Anaconda 또는 Miniconda 설치 시 사용할 수 있으므로,
경량화 버전인 Miniconda를 설치해준다.

설치 방법
https://www.anaconda.com/docs/getting-started/miniconda/install#quickstart-install-instructions

설치를 윈도우 파워셸에서 한 경우라면 conda 명령이 안 먹히는 경우도 있는데
이 경우, 파워셸에서 conda 명령어를 사용할 수 있도록 초기화해야 한다.

아래와 같이 conda.exe 파일이 존재하는 Miniconda 설치 경로를 찾아 초기화해주면 된다.

C:\Users\YourName\Miniconda3\Scripts\conda.exe init powershell

이후 꼭 파워셸을 껐다 다시 실행해주는 것을 잊지 말 것!
파워셸에서 conda 명령어가 인식이 가능해 진다면 다음 ↓ 진행

conda create -n pytorch_tutorial python=3.13 # 가상환경 생성 -n은 네이밍 옵션
conda activate pytorch_tutorial # 방금 생성한 pytorch_tutorial 가상환경 활성화

#코랩에서 실행되는 가상환경 확인하는 법
→ 코랩에서 아래 코드 넣고 실행
import tensorflow as tf
print(tf.__version__)
2.19.0
import torch
print(torch.__version__)
2.8.0+cu126

# 확인된 파이토치 버전 2.8.0+cu126 기준에 맞춰 가상환경 생성
→ 아래는 파이토치 공식 사이트에서 'cpu로만 작동할 pytorch 특정 버전을 설치하고자 할 때'  안내에 따라 찾아낸 명령임

pip install torch==2.8.0 torchvision==0.23.0 torchaudio==2.8.0 --index-url https://download.pytorch.org/whl/cpu

# conda-forge로 라이브러리 설치 
→ 구글에서 "[라이브러리명] conda install"이라 검색하면 믿을만한 공식설치 안내 페이지(anaconda.org 제공) 가 뜬다.
→ -y 옵션도 가능

conda install conda-forge::pandas -y
conda install conda-forge::matplotlib -y
conda install conda-forge::scikit-learn -y

Keras 입력 데이터 타입: tensorflow tensor/np.array
Torch 입력 데이터 타입: torch tensor: np.array + 추가 기능

코드 실행을 위해 아래 명령으로 라이브러리를 추가 설치

pip install timm tqdm

pip와 conda로 설치한 라이브러리 목록은 아래 명령으로 확인 가능

conda list # 무엇으로 설치된 라이브러리인지도 확인 가능

[참고]
윈도 환경은 GPU를 활용한 머신러닝 훈련 효율이 1/2로 줄어든다.
그래서 리눅스를 많이 사용

파이참에서 conda 가상환경을 적용하려면 설정 > 파이썬 > 인터프리터 > 인터프리터 추가
    > 로컬 인터프린터 추가 > 기존 항목 선택 > type: Conda / 환경: pytorch_tutorial 선택
